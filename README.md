# Lido

Solo Bitcoin mining pool for Umbrel.

Dashboard mockup: https://kevinguest.github.io/lido-ui/

## Install

In umbrelOS → App Store → Community App Stores, add:

```
https://github.com/KevinGuest/lido-app
```

Then install **Lido**.

Stratum V1 port: **2301** · Stratum V2 port: **2302** · Web: **2300**
